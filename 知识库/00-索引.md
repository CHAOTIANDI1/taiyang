# 知识库索引

> 这个知识库是给你（零基础用户）查不懂的名词/工具/理念用的。
> AI 不强制读这里，但你跟 AI 协作时遇到任何听不懂的词，告诉我，我会加进来。
>
> **每篇笔记统一标准**：概念 → 功能 → 运作方式（含例子）→ 原理 → 优势 → 使用场景（含对接） → "关联"段（双链到相关笔记）
> **每篇末尾有"关联"段，点链接跳到相关概念，形成知识网**

---

## 使用方法

1. 打开 Obsidian
2. "Open folder as vault"（打开文件夹作为仓库）
3. 选择 `D:\taiyang\知识库\`
4. 左侧出现下面所有笔记
5. **Ctrl+Shift+F 全文搜索任何关键词**
6. 每篇底部"关联"段可以跳转 → 形成知识网

## 这份知识库的网络特性

每篇笔记都有"关联"段，连接其他相关概念。**Obsidian 的 Graph View** 能看到这张网的全貌：
- 打开 Obsidian → 右上角"Open Graph View"
- 你会看到一张像神经元一样的知识网
- 每点多一个笔记 = 网更密一层
- 这就是"知识网状导图"——不是碎片的清单，而是互联的体系

**为什么用网状而不是线性清单**：碎片知识忘得快，连成网的概念忘了 A 看 B 还能想起来 A 是什么——这就是为什么学习要成体系。
