# 30 - 碰撞层（Collision Layer）与碰撞遮罩（Mask）

## 什么是碰撞层？

Godot 的物理引擎用"层"来控制**谁和谁会发生物理碰撞**。

想象一栋楼里有 32 层：
- **layer（自己在哪层）**：你在第几层办公
- **mask（你检测哪层）**：你跟哪层的人有业务往来

只有"A 的 mask 包含 B 的 layer"且"B 的 mask 包含 A 的 layer"时，A 和 B 才会物理碰撞。

## 为什么需要碰撞层？

### 问题场景

俯视角动作游戏里，玩家和怪物都是 CharacterBody2D（物理体）。如果不设碰撞层：

1. **怪物把玩家挤到墙角** → 玩家被夹在怪物和墙之间，所有方向都动不了
2. **怪物互相挤压** → 一群怪挤在一起卡住
3. **hitbox Area2D 检测到自己** → 玩家攻击时自己打自己

### 解决方案：分层

| 层编号 | 层名称 | 谁在这层 | 碰谁（mask） |
|--------|--------|---------|-------------|
| 1 | 墙体层 | 墙、地面、障碍物 | 不碰任何东西（mask=0） |
| 2 | 玩家层 | 玩家 CharacterBody2D | 只碰墙（mask=1） |
| 4 | 怪物层 | 怪物 CharacterBody2D | 只碰墙（mask=1） |

**效果：**
- 玩家碰墙 ✅（玩家 mask=1 包含墙的 layer=1）
- 怪物碰墙 ✅（怪物 mask=1 包含墙的 layer=1）
- 玩家碰怪物 ❌（玩家 mask=1 不包含怪物的 layer=4）
- 怪物碰玩家 ❌（怪物 mask=1 不包含玩家的 layer=2）

**玩家和怪物可以穿过对方**，不会互相挤压。怪物的攻击通过 Area2D 或距离检测来实现，不靠物理碰撞。

## 在 Godot 4 里怎么设置

### 方法 1：在编辑器里设

选中节点 → 右侧检查器 → PhysicsBody2D → Collision → 
- Layer：勾选对应的层
- Mask：勾选要碰撞的层

### 方法 2：在 .tscn 文件里设

```text
[node name="Player" type="CharacterBody2D" parent="."]
collision_layer = 2    ; 玩家在第 2 层
collision_mask = 1     ; 只碰第 1 层（墙）
```

### 方法 3：在代码里设

```gdscript
# 玩家
collision_layer = 2  # 第 2 层
collision_mask = 1   # 只碰第 1 层

# hitbox Area2D 只检测怪物层
_hitbox_area.collision_mask = 4  # 只检测第 4 层（怪物）
```

## 位掩码原理

Godot 的 layer 和 mask 是位掩码（bitmask），不是简单的层号：

| 层 | 十进制值 | 二进制 |
|----|---------|--------|
| 1 | 1 | 00000001 |
| 2 | 2 | 00000010 |
| 3 | 4 | 00000100 |
| 4 | 8 | 00001000 |
| 5 | 16 | 00010000 |

如果要同时在 layer 1 和 3：`collision_layer = 1 + 4 = 5`

## 常见错误

### 错误 1：hitbox 检测到自己
- **原因**：Area2D 没设 mask，默认检测所有层，包括玩家自己的层
- **修复**：设 `_hitbox_area.collision_mask = 4`（只检测怪物层）
- **额外保险**：代码里 `if body == self: continue`

### 错误 2：怪物把玩家挤到墙角卡住
- **原因**：玩家和怪物在同一层，互相物理碰撞
- **修复**：玩家 layer=2 mask=1，怪物 layer=4 mask=1，互不碰撞

### 错误 3：怪物穿墙
- **原因**：怪物的 mask 不包含墙的 layer
- **修复**：确保怪物 mask 包含墙的 layer（mask=1）

## 本项目的碰撞层规范

```
Layer 1（值=1）：墙体、地面、障碍物
Layer 2（值=2）：玩家
Layer 4（值=4）：怪物、NPC
Layer 8（值=8）：道具、可交互物体（将来用）
```

所有物理体的 mask 规范：
- 墙：mask=0（不需要碰任何东西）
- 玩家：mask=1（只碰墙）
- 怪物：mask=1（只碰墙）
- hitbox Area2D：mask=4（只检测怪物）
