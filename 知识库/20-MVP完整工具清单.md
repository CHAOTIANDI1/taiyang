# 20 - MVP 完整工具清单

> 一次性看清点：MVP 阶段你需要什么工具，每个工具干什么、怎么用、多少钱、有没有版权风险。
> 每个按 6 维标准：干什么 / 怎么干 / 优势 / 原理 / 时间成本 / 经济成本 / 版权

---

## A. 开发工具

### 1. Godot 4.x（游戏引擎）

| 维度 | 说明 |
|------|------|
| 干什么 | 把代码/资源变成可运行的游戏 |
| 怎么干 | 拖节点 + 贴 GDScript 脚本 + 跑场景 |
| 优势 | 2D 原生强、轻量 90MB、免费无抽成 |
| 原理 | 可视化节点树 + 自己的脚本语言解释器 |
| 时间 | 1~2 周上手 |
| 经济 | **0 元**（MIT 协议） |
| 版权 | 做出来的游戏完全归你 |

### 2. GDScript（编程语言）

| 维度 | 说明 |
|------|------|
| 干什么 | 写游戏行为代码 |
| 怎么干 | 在 Godot 编辑器里手写或 AI 写你粘 |
| 优势 | 像 Python 容易理解 |
| 原理 | Godot 自己的高级解释型语言 |
| 时间 | 0（AI 写你只需读懂） |
| 经济 | 0 |

### 3. OpenCode + GLM-5（AI 开发助手）

| 维度 | 说明 |
|------|------|
| 干什么 | 写代码 / 设计文档 / 调试 / 决策 |
| 怎么干 | 终端对话 → 它读写文件 → 给方案 |
| 优势 | 命令行原生 + 21 个 skills 治理 |
| 原理 | 调用 GLM-5 大模型 |
| 时间 | 0 学习 |
| 经济 | 按量付费，预计 $50 全程 |

### 4. VS Code（代码编辑器，备选）

| 维度 | 说明 |
|------|------|
| 干什么 | 编辑 .md/.json/.gd 比 Godot 编辑器更好用 |
| 怎么干 | 打开 D:\taiyang → 编辑任何文件 |
| 优势 | 免费、自动补全、Markdown 预览 |
| 时间 | 0 |
| 经济 | 0 |

## B. 协作和管理工具

### 5. GitHub Desktop（版本管理）

| 维度 | 说明 |
|------|------|
| 干什么 | 代码 / 文档存档 + 云端备份 |
| 怎么干 | 三按钮：Commit → 写描述 → Push |
| 优势 | 全世界都在用、个人免费 |

### 6. Obsidian（知识库阅读）

| 维度 | 说明 |
|------|------|
| 干什么 | 看 .md 知识库、双链网络图 |

| 优势 | 本地、双链、不锁定文件 |
| 原理 | 纯看 .md 文件 |
| 时间 | 30 分钟 |
| 经济 | 个人 0 元 |
| 参见 | [[19-Obsidian使用指南]] |

## C. 美术工具

### 7. Gemini（生图）

| 维度 | 说明 |
|------|------|
| 干什么 | 生成角色概念图、怪物图 |
| 怎么干 | 文字描述 → AI 出图 → 保存 PNG |
| 优势 | 谷歌出品，中文 prompt 友好，免费额度够 |
| 原理 | 扩散模型 |
| 时间 | 1 小时能生 10 张候选 |
| 经济 | 个人 0 元 |
| 版权 | ⚠ Google 条款允许商用，但禁止生成有版权角色 |

### 8. Aseprite（像素画+动画）

| 维度 | 说明 |
|------|------|
| 干什么 | 像素画 + 极简帧动画 + Sprite Sheet |
| 怎么干 | 导入 AI 图 → 缩小成 32×32 → 复制帧微调 → 导出长条图 |
| 优势 | 像素画行业标准 |
| 时间 | 1~2 小时基础操作 |
| 经济 | **¥60（Steam 买断）** |
| 替代 | LibreSprite（开源免费） |
| 版权 | 你画/改的图归你 |

### 9. itch.io 免费素材（MVP 占位备用）

| 维度 | 说明 |
|------|------|
| 干什么 | MVP 阶段色块不够看，找现成像素角色/TileSet |
| 怎么干 | [itch.io](https://itch.io/game-assets/free/tag-pixel-art) 筛"free"→ 下载 → 放 Godot |
| 优势 | 现成的 |
| 时间 | 30 分钟 |
| 经济 | 0 |
| 版权 | **看每包具体许可**——多数 CC0 或 CC-BY |

## D. 音乐音效工具

### 10. Suno（BGM 生成）

| 维度 | 说明 |
|------|------|
| 干什么 | 生成游戏 BGM（5 张地图主题曲） |
| 怎么干 | 输入风格描述（如"温暖乡村田园"）→ AI 出 2~4 首候选 → 下满意 |
| 优势 | 不用懂乐理 |
| 原理 | 音乐大模型 |
| 时间 | 1 小时出 5 首候选 |
| 经济 | $10/月（500 首/月，付费版可商用） |
| 版权 | 付费版生成物归你商用 |
| 风险 | 只喂 CC0/自有曲做风格参考避免侵权 |

### 11. ElevenLabs（音效生成）

| 维度 | 说明 |
|------|------|
| 干什么 | 生成一次性音效（剑击、爆炸、水花） |
| 怎么干 | 文字描述"剑打木盾脆响 0.3 秒" → AI 出音效 |
| 优势 | 不用找音效库 |
| 时间 | 几秒一个 |
| 经济 | $5/月（基础包） |
| 版权 | 你生成可商用 |

### 12. 免费音效库（CC0，强烈推荐）

| 来源 | 用途 | 版权 |
|------|------|:---:|
| [freesound.org](https://freesound.org) | 千万 CC0 音效 | CC0 自由商用 |
| [opengameart.org](https://opengameart.org) | 游戏专用 BGM+音效 | 大部分 CC0 |
| [kenney.nl](https://kenney.nl) | Kenney 大神免费音效包 | CC0 |
| [incompetech.com](https://incompetech.com) | Kevin MacLeod BGM | CC-BY 需署名 |

**经济成本：0 元**。CC0 完全自由，CC-BY 需在游戏内"音乐致谢"页署名。

### 13. Audacity（音频编辑）

| 维度 | 说明 |
|------|------|
| 干什么 | 把 AI 生的 mp3 转 .ogg，或改造 CC0 音效 |
| 怎么干 | 导入 → 编辑（剪/调/混响）→ 导出 .ogg/.wav |
| 优势 | 免费开源、行业标配 |
| 时间 | 1 小时上手 |
| 经济 | 0 |
| 版权 | 工具免费，处理素材遵守原素材版权 |

## E. 剧情创作（不换工具）

### 14. OpenCode + brainstorming / deep-interview skill（剧情 agent）

| 维度 | 说明 |
|------|------|
| 干什么 | 探讨剧情方向 / 深挖角色动机 / 写对话 / 设计分支 |
| 怎么干 | 让 AI 加载 brainstorming 或 deep-interview skill 开始对话 |
| 优势 | 不用换工具，21 个 skill 已装好 |
| 原理 | 通用大语言模型（GLM-5）+ 治理 skill 引导 |
| 时间 | 0 |
| 经济 | 0（OpenCode 已有） |
| 版权 | AI 生成的剧情文字归你 |

**为什么不用专用 AI 小说工具**：
- 通用大模型训练数据包含海量小说，剧情创作能力已超过专用工具
- 专用工具（NovelAI、AI Dungeon）是套壳调通用 API，价格更高、效果不一定更好
- 你的 sliver-vibe-coding + brainstorming 已经帮你治理创作流程
