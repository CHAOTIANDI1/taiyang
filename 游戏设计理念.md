## 一、游戏概述

| 项目 | 说明 |
|------|------|
| 游戏名 | **太阳之下（Under the Sun）** |
| 类型 | 实时动作冒险 + 生活模拟 RPG |
| 视角 | 2D / 2.5D 俯视（参考 PokeMMO、星露谷、NDS 风格） |
| 世界观 | 中式玄幻 × 西式奇幻融合 |
| 多人 | 目标为持续性多人世界（类 PokeMMO），MVP 先做单机版 |
| 核心参考 | PokeMMO、宠物王国4烈火、星露谷、大千世界 |

## 二、世界观设定

一个中式玄幻与西式奇幻完全融合的原创世界。不是两块大陆的拼接，而是同一套世界规则下同时存在修仙宗门与魔法学院、东方妖兽与西方魔物、道法与炼金术并存的文化体系。

## 三、战斗系统

### 3.1 操作方式

- 上下左右方向移动，**无翻滚、无连招**
- 不同武器拥有独立普通攻击模组
- **3 个主动技能键**，每个技能有独立的技能攻击模组
- 技能来源：
  - 武器自带技能
  - 护甲附带技能
  - 主职业天赋技能
  - **技能书**：学习指定技能，受职业和属性数值门槛限制
- 技能书获取途径：掉落、商店、任务奖励均可，具体分配待后续细致规划

### 3.2 怪物 AI

每种魔物需编写独立的实时战斗攻击行为逻辑（移动模式、攻击范围、攻击频率、仇恨判断等）。

### 3.3 战斗失败

- 回到上一个存档点 / 主城
- **无死亡惩罚**（不掉落、不扣经验、不扣金币）

## 四、宠物 / 星灵系统

### 4.1 获取方式

| 方式 | 说明 |
|------|------|
| **宠物蛋** | 任务/活动/商店获取，孵化获得宠物 |
| **野外捕捉** | 非直接捕捉——而是魔物有 **低概率** 出现"可捕捉"状态（如 100 只红尾蝎中仅少数可捕捉），需使用 **捕捉器** 道具 |

### 4.2 驯服机制

捕捉成功后，每种魔物需喂食对应的 **特定料理或材料**，才会变得听话（否则可能不服从/不战斗）。不同魔物需求不同。

### 4.3 宠物战斗

| 类别 | 技能数 | 说明 |
|------|--------|------|
| 普通魔物/宠物 | **1 个唯一攻击技能** | 有冷却时间，V1 仅此一种 |
| Boss / 精英怪 | **2~4 个技能** | 后续扩展 |

- 宠物可设置为 **「温和模式」**：不攻击，仅跟随
- 宠物在战斗中由 AI 自动控制，遵循技能冷却
- 宠物暂无进化系统，有 **等级系统**（升级仅增加属性值）

### 4.4 携带与容量

- **携带上限**：1 只
- **拥有上限**：无限（可存储）

## 五、角色成长

### 5.1 属性系统

| 属性 | 全称 | 含义 |
|------|------|------|
| HP | 生命 | 生命值 |
| STR | 力量 | 物理攻击力 |
| MAG | 魔力 | 魔法攻击力 |
| DEF | 防御 | 物理防御 |
| MDEF | 魔防 | 魔法防御 |
| MP | 魔能 | 施法消耗，自然恢复 / 药剂恢复 |

### 5.2 等级体系

- **经验值来源**：击败魔物、完成主线/支线任务、副职业造物产出一并给予角色经验
- **升级方式**：升级后获得可自由分配的 **属性点**，由玩家加点
- **转职方式**：所有角色初始相同（未转职状态），当属性点达到特定门槛后解锁对应主职业的转职资格
- **未转职阶段**：有一套通用基础装备供过渡使用

### 5.3 主职业规划

| 阶段 | 职业 | 说明 |
|------|------|------|
| V1（单机） | **剑士** | 唯一开放职业 |
| 联机版首发 | 剑士 + **法师** + **弓手** | 初始三大职业，共 3 个 |

## 六、副职业系统（每人限选 2 个）

### 6.1 六大副职业

| 序号 | 副职业 | 核心能力 |
|------|--------|----------|
| 1 | **钓鱼姥** | 从地图河流/水域钓出鱼类及各类事物 |
| 2 | **烹饪师** | 用食材和材料烹饪，产出治疗/强化等效果的食物 |
| 3 | **炼金术士** | 材料转化（如"点石成未知材料"需时间）、武器附魔、制造炼金炸弹、拥有专属药剂方 |
| 4 | **锻造师** | 制造武器和护甲，**修复装备耐久度** |
| 5 | **种植师** | 将作物/植物类材料转化为多份种子，种植获得更多产物 |
| 6 | **药剂师** | 用材料调配各类药剂（恢复、强化、功能性药水），调配能力比炼金术士更全面 |

> 各副职业具体能制作什么物品，待主线剧情和动作引擎确定后，在相应阶段逐一明确。

### 6.2 副职业成长

- **升级方式**：制作经验值 + 订单任务
- 不同行为获得经验不同（如钓鱼品质越高经验越多）
- **稀有配方**：达到对应副职业等级才能学习，制作成功给予更高经验
- 具体数值待后续完善

### 6.3 副职业联动

- 钓鱼 → 鱼类可为炼金术士提供材料
- 种植 → 作物可供烹饪师、药剂师使用
- 所有副职业产物均可挂玩家摊位出售
- 锻造师的修复功能对应装备耐久度系统

## 七、物品体系（共 8 类）

| 类别 | 说明 | 可交易 |
|------|------|--------|
| 重要物品 | 剧情用，不可交易、不可丢弃 | 否 |
| 道具 | 通用消耗品、功能道具 | 部分 |
| 武器 | 角色装备，不同武器不同攻击模组 | 是 |
| 护甲 | 角色装备，可附带技能 | 是 |
| 食物 | 烹饪产出，治疗/强化效果 | 是 |
| 材料 | 采集、掉落、副职业用原材料 | 是 |
| 副职业道具 | 副职业专用工具/产物 | 是 |
| 技能书 | 学习技能，受职业和属性数值限制 | 是 |

### 7.1 物品稀有度

白 → 绿 → 蓝 → 紫 → 金

### 7.2 装备耐久度

- 武器/护甲均有耐久度
- 耐久度归零时：**普通攻击仍可用，但装备属性和附带技能全部封禁失效**
- 需找锻造师修复恢复

## 八、剧情与世界结构

### 8.1 主线

- **线性章节制**，主线剧情推进解锁新地图
- 支线任务伴随主线新地图开放同步出现（可一并完成）
- 未来预留：支线解锁地图（非 V1 功能，留接口）
- 对话选项：V1 有对话选项，但不影响结局和剧情走向（剧情确定后可能调整）

### 8.2 叙事 UI

- V1：**纯对话框**（底部文字框）
- 上线后可补充：对话框上方显示角色立绘

### 8.3 地图结构

- 城镇/据点为独立地图区域，以人物为中心移动视角
- 出城方式：走到城边缘的门/路，触发 **黑屏过渡** 进入新地图（类似星露谷进出矿洞、PokeMMO 区域切换）
- 野外地图同理，通过路径/路口连接

### 8.4 怪物分布

- 旅行过程中遭遇：
  - **小怪**：普通敌人
  - **精英怪**：较强敌人
  - **Boss**：剧情 Boss / 野外 Boss

### 8.5 存档机制

| 模式 | 方式 |
|------|------|
| 单机 | 自动存档（进入新地图时触发）+ 手动存档按钮 |
| 联机版 | 实时存档（服务器端持续同步） |

## 九、多人/社交系统

### 9.1 整体定位

- 最终目标：**持续性多人世界**（类 PokeMMO）
- MVP 阶段：**单机版**，先跑通核心操作和剧情
- 多人架构需在单机阶段就预留设计空间

### 9.2 交易市场（频道制）

```
市场频道 1  ←→  市场频道 2  ←→  ...  ←→  市场频道 100
  20 摊位       20 摊位                   20 摊位
```

- 采用 **多频道结构**，类似服务器频道，初始 **1~100 频道**，随玩家增长扩容
- 每个频道容纳 **最多 20 个玩家摊位**
- 摆摊时自动分配：系统按顺序查找下一个有空位的频道（如频道 1 满20、频道 2 已有 10 摊，则分配到频道 2 第 11 号摊位）
- 玩家可自由切换频道进入不同的市场，显示该频道的摆摊玩家
- 每个玩家只能拥有 **1 个摊位**（全局）
- NPC 商人部分随剧情/地图进度变化，单机版必备基础 NPC（贩卖武器和材料）

### 9.3 其他社交功能

- **世界聊天**
- **组队打 Boss**：采用"1P 邀请人世界"方式（参考洛克王国世界），上限 **4 人**
- **PvP 竞技场**：V1 不做，仅留概念

## 十、经济系统

### 10.1 货币

- V1：仅 **金币**
- 未来（多人阶段）：**公会币** 等，暂留概念

### 10.2 金币获取

| 来源 | 说明 |
|------|------|
| 主线/支线任务 | 任务奖励 |
| 副职业产物出售 | NPC 商店 / 玩家摊位 |
| 日常/每周任务 | 稳定金币来源 |
| 副本 | 副本通关掉落金币 |
| 秘境 | 秘境探索掉落金币 |
| Boss 挑战 | Boss 击败掉落金币 |

### 10.3 可交易物

- 材料、食物、锻造装备、副职业产物均可交易
- 重要物品（剧情道具）不可交易

## 十一、游戏循环

### 11.1 主线阶段

接任务 → 野外探险战斗 → 采集材料 → 回城制作 → 推进剧情

### 11.2 主线结束后

- **每日/每周日常任务**（金币来源、材料来源）
- **副本 / 秘境 / Boss 挑战**（重复可玩内容，掉落材料、装备、金币，参考《大千世界》）
- **每周事件活动**

## 十二、家园系统（待确认）

旧版设计中提到"家园实例化 + 世界共享"，结合当前副职业设定，建议如下：

- 每位玩家拥有一个 **私密家园实例**（不与他人共享）
- 家园内设施：
  - **种植田地**：种植师专属，作物按现实时间生长
  - **厨房灶台**：烹饪师使用
  - **炼金台**：炼金术士使用
  - **锻造台**：锻造师使用
  - **药剂台**：药剂师使用
  - **储物箱**：物品存储
- 副职业设施可随副职业等级升级（解锁更多槽位/更高效率）
- 可邀请好友进入家园参观（联机版）

> 请确认或提出你的想法。

---

## 十三、钓鱼系统详细

- 核心机制：**星露谷式钓鱼条**（浮标在鱼图标区域内保持追随）
- **鱼竿品质**影响钓鱼难度
- **鱼饵选择**影响可钓种类和概率
- 副职业等级和解锁的副职业道具可降低钓鱼难度、提高稀有鱼上钩率

---

## 十四、烹饪系统详细

- 需要 **食谱** 才能制作对应食物
- 食谱获取：每日任务、副本、主线剧情，**均有概率掉落/奖励**
- 各食谱所需材料不同

---

## 十五、炼金术士（V1 范围）

- 炼金炸弹（可投掷，范围伤害/异常效果）
- 材料转化（如"点石成未知材料"，需游戏内时间等待）
- 武器附魔
- 专属药剂方（区别于药剂师，品类不同）
- 其他炼金科技待主线和引擎确定后扩展
