# 16 —— 音效与音乐系统

> 我们要为音乐音效打好地基，防止格式不支持、许可纠纷、AI 生成不可商用等问题。本文件是音效系统设计真源。

---

## 资源格式（铁律）

**只用两种格式，永远不踩坑**：

| 格式 | 用途 | 大小 | 为什么 |
|------|------|------|--------|
| `.ogg` (Ogg Vorbis) | BGM 背景音乐 | 1~3 MB/曲 | Godot 原生支持、压缩好、循环无缝 |
| `.wav` (PCM) | 音效一次性短促声 | <100 KB/个 | Godot 原生支持、无解码延迟、命中"叮"立刻响 |

**禁用格式**（不要踩坑）：

| 格式 | 禁用原因 |
|------|---------|
| `.mp3` | Godot 4 支持但循环 seam 不完美，不如 .ogg |
| `.m4a`/`.aac` | Godot 不原生支持，需第三方 |
| `.flac` | 体积大、占内存 |
| `.mid` | 二次创作版权麻烦、Godot 表现不稳定 |

**强制约束写进 AGENTS.md §1.4**：所有音效文件路径必须落在 `data/sounds.json`。

## 数据结构

`data/sounds.json`：

```json
{
  "bgm_village":   { "file": "bgm/village.ogg",   "loop": true,  "volume": 0.6, "category": "bgm" },
  "bgm_field":     { "file": "bgm/field.ogg",     "loop": true,  "volume": 0.6, "category": "bgm" },
  "sfx_sword_hit": { "file": "sfx/sword_hit.wav", "loop": false, "volume": 1.0, "category": "sfx" },
  "sfx_levelup":   { "file": "sfx/levelup.wav",   "loop": false, "volume": 0.8, "category": "sfx" },
  "sfx_fish_splash": { "file": "sfx/splash.wav",  "loop": false, "volume": 0.7, "category": "sfx" },
  "sfx_step_grass":  { "file": "sfx/step_gr.wav","loop": false, "volume": 0.3, "category": "ambient" }
}
```

字段说明：
- `file`：相对 `assets/audio/` 的路径
- `loop`：BGM=true、SFX=false
- `volume`：0~1 音量
- `category`：bgm、sfx、ambient、ui —— AudioManager 按分类统一调节

## AudioManager 单例设计

```
scripts/core/audio_manager.gd  (Autoload 名 AudioManager)
```

接口（仅供参考，实现时再写）：
```gdscript
AudioManager.play_bgm("bgm_village")     # 切 BGM，自动淡入淡出
AudioManager.stop_bgm()                   # 停 BGM
AudioManager.play_sfx("sfx_sword_hit")    # 一次性音效
AudioManager.set_category_volume("bgm", 0.5)  # 分类调音量
```

底层内部维护：
- 1 个 `AudioStreamPlayer` 给 BGM
- 1 个对象池给 SFX（避免频繁新建销毁）
- 读 `data/sounds.json` 配置

**地基规矩 1（逻辑显示分离）**：AudioManager 只管"播什么声音、播多大、什么时候停"——**不直接耦合战斗逻辑/物品逻辑**。其他系统发指令"播 music_id"，AudioManager 自己怎么播自己定。

## 资源获取（三种来源）

### A. 免费 CC0 音效库（推荐 MVP）

| 来源 | 用途 | 许可 | 修改入游戏 |
|------|------|------|:---:|
| [freesound.org](https://freesound.org) | 音效 | CC0（部分 CC-BY） | ✅ 大部分 |
| [opengameart.org](https://opengameart.org) | BGM + 音效 | CC0/CC-BY/GPL | ✅ |
| [kenney.nl](https://kenney.nl) | 音效包 | CC0 | ✅ |
| [incompetech.com](https://incompetech.com) | BGM | CC-BY 4.0 | ✅（署名 Kevin MacLeod） |
| [Pixabay Music](https://pixabay.com/music) | BGM | Pixabay 许可（类似 CC0） | ✅ |

### B. AI 生成音乐

| 工具 | 类型 | 免费 | 商用 | 推荐场景 |
|------|:---:|:---:|:---:|---------|
| **Suno** | AI 全曲生成 | 试听免费，商用付费订阅 | ✅ 付费版可商用 | 新手村/主城 BGM |
| **Udio** | AI 全曲生成 | 同上 | ✅ 付费版 | Boss 战 BGM |
| **AIVA** | AI 作曲 | 免费有额度 | ✅ 付费版 | 情感向 BGM |
| **MusicFX (Google)** | AI 生成 | 免费 | 限制商用 | 试验 |

**MVP 推荐 Suno**：付费约 $10/月可生成 500 首商用音乐。

### C. CC0 音效用 AI 改造为"自己的"

**合法路径**：
1. 从 freesound.org 下载一个 CC0 剑击音效（自由使用无署名）
2. 用 AI 音效工具改造（比如加混响、变调、拼接）—— Audacity + AI 插件，或 Adobe Podcast AI Enhance
3. 改造后保存为自己的 .wav
4. 版权：**仍 CC0**，你不存在需要署名问题，可视为你自己的

**对 BGM 也一样**：
1. 从 Pixabay 下"乡村田园" BGM（CC0-like 许可）
2. 用 Suno 的"风格参考"功能喂原曲，生成同风格变体
3. 改后存为 .ogg
4. 版权：Suno 付费版生成物归你所有，原曲 CC0 不需署名

**风险点**：Suno 喂现有曲做风格参考时**最好喂 CC0 曲**，避免喂受版权保护曲触发 Suno 服务条款的"不能输出侵权作品"。已记录为风险。

## 地基风险点

1. **Suno 喂曲版权**：只喂 CC0 或自有曲做参考，不喂受版权保护的流行曲
2. **下载素材署名**：CC-BY（必须署名）的素材，使用时要在游戏内"音乐致谢"页面署名
3. **格式强制**：所有音频入库前必须转 .ogg / .wav，禁用其他格式
4. **音量缩放策略**：所有 SFX 不超 0dB（避免爆音），通过 AudioManager 统一调
5. **资源体积**：BGM 单曲 < 3 MB，SFX < 100 KB，避免包体膨胀
