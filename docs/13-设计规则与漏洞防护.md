## 20 条漏洞防护

### 战斗漏洞

| # | 漏洞 | 处理 |
|---|------|------|
| 1 | 食物战斗无 CD → 永生 | 战斗中 15 秒 CD + 吃东西动画 |
| 2 | 远程白嫖近战怪 | 3 秒脱战回原位 + 回满血 |
| 3 | Boss 进出刷新 | 离 Boss 区域 → Boss 回满血 |
| 4 | 温和模式宠物被打死 | 温和 = 无敌 + 不可被选中 |
| 5 | 贴脸攻击穿模 | Hitbox 比视觉稍大 |

### 经济漏洞

| # | 漏洞 | 处理 |
|---|------|------|
| 6 | 无限刷怪产金币 | 联机版每日掉落上限 |
| 7 | NPC 套利 | 产物售价 < 材料成本 × 60~80% |
| 8 | 市场无价格限制 | 按稀有度设价格区间 |
| 9 | 重要物品误卖 | 代码层禁止出售 |

### 系统漏洞

| # | 漏洞 | 处理 |
|---|------|------|
| 10 | 改系统时间作弊 | 不读系统时钟，记时间戳对比 |
| 11 | 手动覆盖自动存档 | 不同槽位，互不覆盖 |
| 12 | 组队抢装备 | 独立掉落 |
| 13 | 副本无限刷 | 每日次数限制 + 门票制 |
| 14 | 技能书跨职业 | 三个标签：职业/等级/属性 |
| 15 | 刷属性点 | 三个来源唯一：升级/主线/道具 |

### 交互漏洞

| # | 漏洞 | 处理 |
|---|------|------|
| 16 | 世界聊天剧透 | 留举报按钮，审核后续做 |
| 17 | NPC 对话被打断 | 对话期角色无敌 + 不可移动 |
| 18 | 副职业忘了学 | 职业凭证道具 |
| 19 | 技能书学完还在 | 消耗品，学完消失 |
| 20 | 物品栏满奖励被吞 | 奖励无视容量，满包发邮箱 |

## 优先级表

| 优先级 | 漏洞编号 |
|--------|---------|
| **P0（MVP 必做）** | #1, #4, #9, #15, #20 |
| **P1（MVP 之后）** | #2, #5, #7, #10, #11 |
| **P2（技能书系统启用时）** | #14, #19 |
| **P3（联机版）** | #3, #6, #8, #12, #13, #16 |
| **P4（长期优化）** | #17, #18 |

> **变更说明**：#14（技能书职业门槛）和 #19（技能书消耗品）从 P0 移到 P2——因为 MVP 不实现技能书系统，对应漏洞防护在技能书系统启用时同步处理。
