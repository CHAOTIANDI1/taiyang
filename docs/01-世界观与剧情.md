# 01 —— 世界观与剧情

---

## 世界观

一个中式玄幻与西式奇幻完全融合的原创世界。不是两块大陆的拼接，而是同一套世界规则下同时存在修仙宗门与魔法学院、东方妖兽与西方魔物、道法与炼金术并存的文化体系。

世界传说：太阳是万物之源。远古一位"追日者"追逐太阳获得了力量，建立了失落的远古文明。如今远古文明已没落，但太阳依然高悬，被称为"太阳之下"。

## 前情故事（开场独白/白屏文字）

我是一名社畜——这个词现在说起来不算刺耳，毕竟它已经很贴切了。

我曾也有过梦想。也曾努力奔跑。

但世事不如意的存在总是会让人意识到一些事：买房的首付遥遥无期，房价的水涨船高；长时间的高压状态一点点蚕食掉我曾经热络的热忱；不规律的作息和饮食也在悄悄替我的身体记账。

我曾以为——努力就能变得更好。后来我明白——这只是一种"不努力就会被生活淘汰"的人生马拉松。

幸好，今天是休息日。

属于我自己的，一次停下来思考的机会。

上班的生物钟还是六点准时叫醒了我就被生物钟准时拉了起来。冥冥之中，我走上楼顶天台，想看看初升的太阳，看看这座在周日里渐渐沉寂下来的城市。

我还记得小时候，特别喜欢追着太阳跑；喜欢把五指张开挡在太阳前，让阳光像母亲一样从我指缝里漏下来照在我脸上——好温暖。

这一刻，我的心很宁静。那些杂七杂八的念头在初升的阳光里一点一点沉下去，慢慢安静了。

然后我好像看见——太阳离我越来越近。

是错觉吧？

突然就有了一个念头：我想睡一觉。做一个甜蜜的梦。梦一个我内心真正向往的人生。一个我可以重新开始的梦。

我缓缓闭上了眼睛——好奇妙。好温暖。这一次我的心更安宁，安宁得像被什么按住了所有的杂念。

再一睁眼——

我正坐在一座老旧房屋的屋顶上，屁股下那红红的瓦片烫得我"嘶"地吸了口气。

这里是我的家？……这里是我的家。

这是我在这个世界的起点。是我的一段新人生。

一切，从**太阳之下**开始。

## 主线结构

- **线性章节制**，主线剧情推进解锁新地图
- 支线任务伴随主线新地图开放同步出现（可一并完成）
- 未来预留：支线解锁地图（非 V1 功能，留接口）
- 对话选项：V1 有对话选项，但不影响结局和剧情走向（剧情确定后可能调整）

**主题核心**：太阳 = 希望/温暖/新生
- 现实世界里太阳已无法温暖社畜的心
- 游戏世界里太阳具有真正的力量，但正在经历危机
- 玩家身上的太阳印记是关键

## 章节大纲

### 第一章：初升之阳（MVP 实现）

| 节点 | 内容 |
|------|------|
| 开场 | 黑屏独白（前情故事）→ 醒来坐在屋顶 |
| 新手村 | 引路人 NPC 解释世界基本情况，引导基础教学（移动、战斗） |
| 第一个宠物蛋 | 完成新手村任务获得第一颗宠物蛋，孵化宠物 |
| 第一只可捕捉怪 | 在野外遇到一只头顶闪着金光的"可捕捉"魔物（教程提示） |
| 子职业试玩 | 钓鱼体验、烹饪体验，引导选择 1 个副职业 |
| 主城 | 到达主城，转职引导——剑士职业导师，传授转职后技能书 |
| 小型副本 | 探索主城外的第一个野外地图片区，遭遇小怪+精英怪 |
| 第一章 Boss | 一只被"暗影印记"侵蚀的野生精英怪威胁新手村，玩家击败 |
| 太阳印记 | Boss 死后身上出现光芒，玩家接触后印记出现在自己身上 |
| 第一章结语 | 引路人认出印记与"追日者"传说相关，告知玩家可能是传承者，但具体未知。玩家初步掌握自己在世界中的位置。 |

### 第二章：双月之争（后续版本）

- 主城两势力：日之（秩序/温暖）与 月之（冷静/理性）暗中冲突
- 玩家中立穿行两势力，接副职业进阶、配方解锁、宠物食材订单
- 章末发现冲突由第三方"暗日教"挑动
- 第二章 Boss：暗日教核心祭司

### 第三章：最后之日（后续版本）

- 真相揭示：太阳正在缓慢熄灭，源于远古"偷日事件"，暗日教前身窃取部分太阳之心
- 玩家身上的太阳印记是失落已久的太阳碎屑，是复活太阳的关键
- 主城附近出现暗影怪潮
- 第三章 Boss：暗日教教主

### 第四章：太阳之下（后续版本）

- 玩家集齐碎屑，面对选择：自我牺牲复活太阳 vs 成为新一代追日者
- 结局：玩家将碎屑融入印记，成为追日者，太阳因人心而亮，不再依赖神力
- 主题回响：阳光下不被生活淘汰——呼应玩家现实经历
