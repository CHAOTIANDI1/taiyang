# 01 —— 世界观与剧情

---

## 世界观

一个中式玄幻与西式奇幻完全融合的原创世界。不是两块大陆的拼接，而是同一套世界规则下同时存在修仙宗门与魔法学院、东方妖兽与西方魔物、道法与炼金术并存的文化体系。

世界传说：太阳是万物之源。远古一位"追日者"追逐太阳获得了力量，建立了失落的远古文明。如今远古文明已没落，但太阳依然高悬，这个世界被称为"太阳之下"。

---

## 核心谜题（贯穿全剧的悬疑线）

| 谜题 | 状态 |
|------|------|
| 这个世界真实存在，还是只在主角想像中？ | 留疑点，后续定 |
| 主角的失忆原因 | 留疑点，后续定 |
| 主角身上为什么没有印记 | 留疑点，后续定 |
| 印记到底是什么 | 留疑点，后续定 |

**核心机制**：穿越者都有印记，印记包含穿越后世界 + 原世界**两个世界的全部记忆**。主角没有印记 = 主角丢失了异世界的记忆。

**第二章主线**：主角围绕"找太阳印记"推进剧情——目的是通过印记找回异世界（这个世界）的记忆。

---

## 穿越者法则

- 所有穿越者身上都**自带印记**
- 印记包含*两个世界*的全部记忆（现实世界 + 异世界）
- 主角特殊：穿越过来但**没有印记**（或印记未显现）
- 主角只记得现实世界"社畜"部分，异世界的记忆丢失
- 主角刚搬来此地一天，设定为"新人"

### 前情独白文本

我是一名社畜——这个词现在说起来不算刺耳，毕竟它已经很贴切了。

我曾也有过梦想。也曾努力奔跑。

但世事不如意的存在总是会让人意识到一些事：买房的首付遥遥无期，房价的水涨船高；长时间的高压状态一点点蚕食掉我曾经热络的热忱；不规律的作息和饮食也在悄悄替我的身体记账。

我曾以为——努力就能变得更好。后来我明白——这只是一种"不努力就会被生活淘汰"的人生马拉松。

幸好，今天是休息日。

属于我自己的，一次停下来思考的机会。

上班的生物钟六点准时叫醒了我。冥冥之中，我走上楼顶天台，想看看初升的太阳，看看这座在周日里渐渐沉寂下来的城市。

我还记得小时候，特别喜欢追着太阳跑；喜欢把五指张开挡在太阳前，让阳光像母亲一样从我指缝里漏下来照在我脸上——好温暖。

我看了一眼手表——6 点 26 分。

这一刻，我的心很宁静。那些杂七杂八的念头在初升的阳光里一点一点沉下去，慢慢安静了。

然后我好像看见——太阳离我越来越近。

是错觉吧？太阳里好像有一个发光的符号。

突然就有了个念头：我想睡一觉。做一个甜蜜的梦。梦一个我内心真正向往的人生。一个我可以重新开始的梦。

我缓缓闭上了眼睛——好奇妙。好温暖。这一次我的心更安宁，安宁得像被什么按住了所有的杂念。

再一睁眼——

我正坐在一座老旧房屋的屋顶上，屁股下那红红的瓦片烫得我"嘶"地吸了口气。

这里是我的家？……这里是我的家。

这是我在这个世界的起点。是我的一段新人生。

一切，从**太阳之下**开始。

## 主题核心

太阳 = 希望 / 温暖 / 新生

- 现实世界：太阳已无法温暖社畜的心
- 游戏世界：太阳具有真正的力量，但正在经历危机
- 玩家身上的太阳印记是关键线索

**结局方向**：暂不定。第 3 章制作时再决定最终结局走向（是"接受并放下现实"还是"重新热爱现实"）。MVP 只做第一章。

## MVP 地图结构（第一章）

共 **5 张地图**，按主线顺序解锁：

```
① 玩家的家（院子+屋内）
    │  从院子边缘出口走 →
    ▼
② 新手村（小村落，含教程 NPC、家入口、村长）
    │  从村边缘出口走 →
    ▼
③ 野外 A（草原/小径，小怪出没，河边钓鱼点）
    │  继续走 →
    ▼
④ 野外 B（森林边缘，精英怪出没）
    │  继续走 →
    ▼
⑤ 小城镇（含 NPC 商店、铁匠、转职导师、烹饪 NPC）
```

### 地图详细

| # | 地图名 | 类型 | 内容 |
|---|--------|------|------|
| ① | 玩家的家 | 私密实例 | 院子（烹饪灶台位、未来其他设施位）+ 屋内（床/存档点、储物箱）。从院子边缘走到出口 → 新手村。进屋触发渐黑过渡。 |
| ② | 新手村 | 公共地图 | 3~4 个 NPC（村长/引导人、杂货商、钓鱼向导、邻居）。家入口在村内。边缘 2 个出口：一个回家、一个去野外 A。未来联机版可见其他玩家。 |
| ③ | 野外 A·晴原小径 | 野外地图 | 草原风格。2 种小怪出没。路边有钓鱼点（河流）。钓鱼向导 NPC 站在河边。 |
| ④ | 野外 B·林暗边缘 | 野外地图 | 森林风格。1 种小怪 + 1 种精英怪。野外 Boss 出没点。边缘出口通向小城镇。 |
| ⑤ | 小城镇 | 城镇地图 | 更大的聚落。NPC 商店（卖白/绿装）、NPC 铁匠（修复装备）、转职导师、烹饪副职业 NPC。此处不是未来玩家交易大城镇，是第一章的城镇据点。 |

## 第一章：初升之阳（完整剧情草稿）

| 节点 | 时段 | 剧情 |
|------|------|------|
| 1. 开场 | 0~3min | 漫画/文字独白 → 坐在自家屋顶 → 瓦片烫屁股 → 爬下来 |
| 2. 玩家的家 | 3~5min | 院子里转一圈。提示存档点（床）。灶台提示"需要一个烹饪师才能使用"。从院子出口走向新手村，触发第一次渐黑过渡。 |
| 3. 新手村·相遇 | 5~10min | 进入新手村。村长 NPC 在村口迎接，惊讶"你刚搬来那座老宅？才一天就起来了？"引导玩家移动教学（上下左右）。杂货商 NPC 教基础道具使用。 |
| 4. 新手村·战斗教学 | 10~20min | 村长引导到村边缘试炼场。出现一只训练用小怪，教学攻击操作、3 个技能键、前摇识别。击败后获得第一个宠物蛋。 |
| 5. 宠物孵化 | 20~25min | 回到院子，用蛋孵化台（或床旁）孵化宠物。宠物破壳而出，自动跟随。首次战斗体验。 |
| 6. 野外 A·探索+钓鱼 | 25~40min | 从新手村另一出口走向野外 A。沿途遇野生小怪 2 种。河边钓鱼向导 NPC 站着，引导钓鱼体验任务（第一次副职业体验）→ 正式学钓鱼。 |
| 7. 野外 B·精英 | 40~50min | 继续走到野外 B。遇到精英怪，战斗后 NPC 路人（原生居民）受伤求助 → 引导主角到小城镇。 |
| 8. 小城镇·转职 | 50~60min | 进入小城镇。转职导师引导——显示属性面板、转职门槛。完成转职为剑士。转职导师从身上取下一把旧剑："这把剑跟了我三十年……它等到了对的人。"获赠一套绿色剑士装备。 |
| 9. 小城镇·烹饪 | 60~75min | 烹饪 NPC 引导烹饪体验任务（第二次副职业体验）→ 正式学烹饪。两个副职业均完成学习。 |
| 10. 灾厄降临 | 75~90min | 村长托人传来消息：新手村附近出现异常暗影怪潮，疑似有一只被"暗影印记"侵蚀的精英怪在作祟。玩家赶回野外 B 区域。 |
| 11. 第一章 Boss 战 | 90~100min | 野外 B 某处出现 Boss（曾是穿越者，被暗影侵蚀异化的精英怪）。3 阶段 Boss 战。Boss 临死说几句话（台词暂定）。Boss 死后掉落一个印记（重要物品，不可丢/卖）。主角在开场漫画中见过这个符号，下意识捡起 → 一阵光芒铺满屏幕 → 漫画分镜展示 Boss 的前世今生（涉及下方的城市，具体内容留空待做 Boss 战时再定）。 |
| 12. 第一章结语 | 100~110min | 回到小城镇。村长/转职导师认出太阳印记与"追日者"传说相关。印记"拨开了主角一些记忆"，但记忆仍不完整。村长提示"穿越者都带印记，但你的印记是后得的……很奇怪"。主角确定要围绕太阳印记继续探索。对话结束，主线第一章完成。**MVP 到此为止。** |

### 第一章要点

- 钓鱼在野外 A（节点 6）先学，烹饪在小城镇（节点 9）后学
- 主角设定为"刚搬来一天的新人"，异世界记忆已忘，只记得社畜经历
- Boss 是曾被暗影侵蚀的穿越者（不是原生怪），主角通过 Boss 掉落的印记看到 Boss 的一生
- 印记是**重要物品**（不可丢/卖/交易），是推动剧情的关键道具
- Boss 一生涉及"下方的城市"（为第二章铺垫，具体内容待做时再定）
- 印记在 MVP 中拾取后可看 Boss 一生（漫画分镜/黑屏文字暂代），但**无属性效果**
- 主角"没有印记"是世界观的核心疑点，第二章开始解谜
