# 11 —— MVP 范围定义

> Version 0.1 最小可玩版本。验证核心手感，再扩展。

---

## MVP 包含

| 内容 | 范围 |
|------|------|
| 地图 | 1 新手村 + 1 野外地图片区 + 1 主城（含 NPC 商店） |
| 武器 | 剑 1 把（剑士） |
| 怪物 | 3~4 种（2 小怪 + 1 精英 + 1 Boss） |
| 宠物 | 宠物蛋获取 ×3 种（孵化后战斗） |
| 副职业 | **只做 2 个**：钓鱼 + 烹饪 |
| 等级 | 1~10 级 |
| 任务 | 主线第一章（3~5 个任务） |
| 联机 | 不做，仅留接口 |

---

## MVP 不包含

| 内容 | 原因 |
|------|------|
| 法师/弓手职业 | V1 只做剑士 |
| 炼金/锻造/种植/药剂副职业 | 只做钓鱼+烹饪 |
| 交易市场频道系统 | 联机功能，MVP 只 NPC 商店 |
| 世界聊天 / PvP / 组队 | 联机功能 |
| 副本/秘境系统 | 主线结束后内容 |
| 季节/天气 | 太复杂，预留字段 |
| 宠物进化 | 只有等级系统 |
| 技能书全途径 | 预留，先做武器/护甲/职业自带技能 |
| 角色立绘 | 纯对话框 |
| 后端/数据库 | 单机版用本地存储 |

---

## MVP 验证目标

```
角色移动手感是否流畅？ → 走路 + 方向响应
战斗是否"看得懂"？ → 怪物前摇 + 攻击判定 + 受击反馈
宠物 AI 是否不碍事？ → 跟随 + 攻击 + 防卡死
钓鱼是否有趣？ → 钓鱼条 + 鱼竿品质 + 鱼饵
烹饪是否有意义？ → 食谱 + 材料 + 效果应用
```

---

## 开发路线

```
第 1 步（现在）：纯 MVP → 验证核心手感（2~4 周）
第 2 步（验证后）：缩小版全内容 → 接入所有系统极简版（4~8 周）
第 3 步（全程并行）：边做边讨论下一个系统
```
