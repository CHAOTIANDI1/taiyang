# 07 —— 地图与世界结构

---

## 地图结构

- 城镇/据点为独立地图区域，以人物为中心移动视角
- 出城方式：走到城边缘的门/路，触发**黑屏过渡**进入新地图
  - 实际实现：渐黑（0.5 秒）→ 短暂加载 → 渐亮（0.5 秒）→ 新地图名居中飘入再消失（1~2 秒）
- 野外地图同理，通过路径/路口连接
- 主线解锁新地图，支线伴随主线开放

---

## 存档机制

| 模式 | 方式 |
|------|------|
| 单机 | 自动存档（进入新地图时触发）+ 手动存档按钮 |
| 联机版 | 实时存档（服务器端持续同步） |

### 存档保护

- 3~5 个**手动存档槽**（玩家自选位置）
- 1 个**独立自动存档槽**（进新地图触发）
- 自动存档不覆盖手动存档
- 手动存档槽只被玩家覆盖
- **战斗中禁用手动存档**（防止必死瞬间存档）
- 读档时如果血量低于 20%，自动把玩家送到最近主城

---

## 死亡传送

- 传送到当前地图的安全点坐标（不是直线距离最近的主城）
- Boss 战死亡 → Boss 门外重生

## 家园系统

每位玩家拥有一个**私密家园实例**（不与他人共享）。

### 家园设施

| 设施 | 用途 |
|------|------|
| 种植田地 | 种植师专属，作物按现实时间生长 |
| 厨房灶台 | 烹饪师使用 |
| 炼金台 | 炼金术士使用 |
| 锻造台 | 锻造师使用 |
| 药剂台 | 药剂师使用 |
| 储物箱（仓库） | 物品存储，可升级扩容 |
| 钓鱼池（可选） | 可选设施 |

- 副职业设施可随副职业等级升级（解锁更多槽位/更高效率）
- 可邀请好友进入家园参观（联机版）
