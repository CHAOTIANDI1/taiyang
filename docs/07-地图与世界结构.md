## 玩家的家

家是新手村旁边的一个**独立地图**，从新手村的某个入口（小路）进入触发渐黑过渡。

### 家的构成（一张地图，两个子区域）

```
┌─────────────────────────────────┐
│         院 子（屋外）             │
│  ┌─────────┐                    │
│  │ 烹饪灶台 │   ← MVP 实现       │
│  └─────────┘                    │
│  ┌─────────┐                    │
│  │ 未来设施 │   ← 预留位置，不实现 │
│  │  种植田  │                    │
│  │  炼金台  │                    │
│  │  锻造台  │                    │
│  └─────────┘                    │
│                    ┌──────┐     │
│                    │ 屋门 │→ 渐黑→ 屋内 │
│                    └──────┘     │
│  ┌────────────────────┐         │
│  │ 院子出口 → 新手村    │         │
│  └────────────────────┘         │
└─────────────────────────────────┘

屋内（渐黑过渡后进入）：
  - 床（存档点）
  - 储物箱（仓库，可升级扩容）
  - 装饰/家具（未来扩展）
```

- 院子与屋内是**两张地图**，走门触发渐黑过渡
- 院子是副职业设施的集中地，烹饪灶台 MVP 实现，其余预留位置
- 屋内只有床 + 储物箱，未来可加装饰

## 地图过渡

- 走到地图边缘的出口（门/路/小径）→ 触发**渐黑过渡**
- 渐黑（0.5 秒）→ 短暂加载 → 渐亮（0.5 秒）→ 新地图名居中飘入再消失（1~2 秒）
- 城镇/据点以人物为中心移动视角
- 野外地图同理，通过路径/路口连接
- 主线解锁新地图，支线伴随主线开放

## 存档机制

| 模式 | 方式 |
|------|------|
| 单机 | 自动存档（进入新地图时触发）+ 手动存档按钮（床/暂停菜单） |
| 联机版 | 实时存档（服务器端持续同步） |

### 存档保护

- 3~5 个**手动存档槽**（玩家自选位置）
- 1 个**独立自动存档槽**（进新地图触发）
- 自动存档不覆盖手动存档
- 手动存档槽只被玩家覆盖
- **战斗中禁用手动存档**（防止必死瞬间存档）
- 读档时如果血量低于 20%，自动把玩家送到最近主城

---

## 死亡传送

- 传送到当前地图的安全点坐标（不是直线距离最近的主城）
- 每个地图标记自己的安全点
- Boss 战死亡 → Boss 门外重生
