# 06 —— 物品体系

> 共 8 类。重要物品无限容量，家园仓库可升级扩容。

---

## 物品类别

| 类别 | 说明 | 可交易 | 容量 |
|------|------|:---:|------|
| 重要物品 | 剧情用，不可交易、不可丢弃 | 否 | **无限** |
| 道具 | 通用消耗品、功能道具 | 部分 | 有限 |
| 武器 | 角色装备，不同武器不同攻击模组 | 是 | 有限 |
| 护甲 | 角色装备，可附带技能 | 是 | 有限 |
| 食物 | 烹饪产出，治疗/强化效果 | 是 | 有限 |
| 材料 | 采集、掉落、副职业用原材料 | 是 | 有限 |
| 副职业道具 | 副职业专用工具/产物 | 是 | 有限 |
| 技能书 | 学习技能，受职业和属性数值限制 | 是 | 有限 |

---

## 稀有度

白 → 绿 → 蓝 → 紫 → 金

---

## 装备耐久度

- 武器/护甲均有耐久度
- 耐久度归零时：**普通攻击仍可用，但装备属性和附带技能全部封禁失效**
- 需找锻造师修复恢复
- 装备**永不永久损坏**
- 耐久消耗速度：普通打怪约 2~3 小时耗尽一件
- 修复费用分档：
  - 白/绿装：免费修（保护新手）
  - 蓝装：少量金币
  - 紫/金装：较贵但绝对可修
- 耐久条在装备图标上可见，归零时弹出提示

---

## 技能书规则

- 技能书带三个标签：`[需求职业]` `[需求等级]` `[需求属性]`
- 不满足标签的不允许使用，描述中清晰显示
- 技能书使用后**立即消失**（消耗品逻辑）
- 已学会该技能则按钮变灰显示"已学会"
- 技能书可交易
