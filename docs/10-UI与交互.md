## 奖励发放

- 任务奖励无视背包容量
- 普通奖励满包时发到**临时邮箱/暂存箱**（24 小时限领）
- 重要物品直接越过容量检查发放

## 邮件系统 UI（MVP 简化版）

| 功能 | MVP | 联机 |
|------|:--:|:--:|
| 邮件列表显示 | ✅ | ✅ |
| 邮件附件领取 | ✅ | ✅ |
| 邮件过期 24h 倒计时显示 | ✅ | ✅ |
| 红点提示未读邮件 | ✅ | ✅ |
| 玩家间邮件 UI | ❌ | ⏳ |
| GM 群发 UI | ❌ | ✅ |

### MVP 邮件流程

```
玩家满包时获得奖励 → 自动入邮件 → 邮箱红点提示
→ 玩家打开邮箱 → 看到邮件 + 附件 → 点领取 → 附件入包 → 邮件标记已读
→ 24h 后邮件过期删除（附件未领则提示丢失）
```

数据配置走 `data/mails.json`，逻辑走 `scripts/core/mail_manager.gd`。
